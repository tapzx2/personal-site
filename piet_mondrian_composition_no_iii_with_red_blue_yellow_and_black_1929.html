Mondrian Project

Recreating Composition No. III, with Red, Blue, Yellow, and Black, 1929 with CSS Grid

Assignment

  • Recreate Composition No. III using CSS Grid
  • Make it responsive

Process

The assignment was inspired by the Mongrid project.

  1. Map the dimensions
    I laid a Photoshop grid over the painting. This would serve as my model for what the CSS Grid should be. I smallest grid lines to the smallest width of any shape (16.5) and the largest to 132px. If I were to do this again I would have set largest to 165.

  2. Translate to Grid
    On a 100 x 100 grid, going left to right I filled grid column/row start/end for each shape.
    grid-column-start: 1; grid-column-end: 43; grid-row-start: 1; grid-row-end:55;

  3. Make it responsive
    I was having trouble doing this with CSS and Grid. So, I decided to use Javascript.

    const grid = document.querySelector(".mondrian-wrapper") function resizeGrid() { grid.style.height = ${grid.clientWidth}px } //on load set height to width resizeGrid(); //on window resize window.addEventListener('resize', resizeGrid);

    I don't know if this is the accepted way to do this, but it worked in the browser and mobile.

  4. Fix the colors
    I updated the grid colors by sampling the hex codes from original image. I also set the colors to be CSS variables. I love variables so much.

Reflection

I didn't watch Mongrid project video before starting. I probably should have. In all, it took me 2.5 hrs to recreate Composition III in CSS Grid and another 20 minutes to plug it into my portfolio site. I had fun doing this project!
