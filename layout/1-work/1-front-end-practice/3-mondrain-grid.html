Mondrian Grid

A classic grid project from Jen Simmons. My implementation was a little different. I laid a Photoshop grid over the painting to create a more accurate replica of Composition No. III, with Red, Blue, Yellow, and Black, 1929. Also, I was having trouble making the grid responsive with just CSS. I decided to use Javascript. 


      const grid = document.querySelector(".mondrian-wrapper"); 

      function resizeGrid() { grid.style.height = `${grid.clientWidth}px` } //on load set height to width

      resizeGrid(); // on window resize

      window.addEventListener('resize', resizeGrid);
